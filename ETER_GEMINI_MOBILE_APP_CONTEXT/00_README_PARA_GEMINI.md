# Paquete oficial de contexto para Gemini

Esta carpeta es un paquete de contexto limpio para crear una app mobile-first de ETER. No es una app, no contiene credenciales y no debe ejecutarse como proyecto.

Usar estos archivos como fuente oficial para entender marca, producto, catalogo, revendedores, stock, pantallas, integraciones y reglas de seguridad. Si una informacion no aparece aca, no inventarla: marcarla como `PENDIENTE_DE_CONFIRMAR`.

Orden recomendado de lectura:

1. `01_MASTER_CONTEXT_ETER.md`
2. `02_BRAND_IDENTITY_AND_TONE.md`
3. `03_EXISTING_SITE_STRUCTURE.md`
4. `04_MOBILE_APP_REQUIREMENTS.md`
5. `05_CATALOG_DATA_MODEL.json`
6. `06_RESELLER_SYSTEM_RULES.md`
7. `07_AI_TOOLS_SPEC.md`
8. `08_GOOGLE_INTEGRATIONS_SPEC.md`
9. `09_UI_SCREEN_BLUEPRINTS.md`
10. `10_ASSETS_MANIFEST.json`
11. `11_GEMINI_FINAL_PROMPT.md`
12. `12_IMPLEMENTATION_NOTES_FOR_ME.md`

Objetivo final: construir una aplicacion movil basada en ETER, orientada a catalogo, stock real, revendedores, herramientas IA, login y gestion privada, manteniendo el tono directo de Mar del Plata y la estetica dark/neon de la marca.
