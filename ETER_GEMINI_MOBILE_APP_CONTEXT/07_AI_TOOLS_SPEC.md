# Herramientas IA para la app

Todas las herramientas deben usar contexto oficial ETER, no inventar stock/precios y pedir confirmacion antes de acciones externas.

## Generador de mensajes WhatsApp

- Sirve para: responder clientes, pasar info de producto, pedir datos, confirmar stock.
- Entrada: producto, talle, tono, consulta del cliente.
- Salida: mensaje breve listo para copiar.
- Prompt interno: "Sos Aura, asistente oficial de ETER. Escribi en voseo argentino, directo, sin prometer stock. Si falta dato, pedi confirmacion."
- Pantalla: Herramientas IA y detalle de producto.
- Login: recomendado para revendedores.

## Redactor Gmail

- Sirve para: preparar emails a clientes, proveedores o equipo.
- Entrada: destinatario, objetivo, datos, tono.
- Salida: asunto y cuerpo de email como borrador.
- Prompt interno: "Redacta un Gmail profesional y claro para ETER. No envies nada. Marca datos faltantes como PENDIENTE_DE_CONFIRMAR."
- Pantalla: Herramientas IA > Gmail.
- Login: si.

## Generador de publicaciones

- Sirve para: crear copy para Instagram/WhatsApp estados.
- Entrada: producto, talles, beneficio, CTA.
- Salida: variantes cortas.
- Prompt interno: "Crear copy ETER sin emojis innecesarios, sin promesas de ganancia, sin stock garantizado."
- Pantalla: Detalle producto y Herramientas IA.
- Login: recomendado.

## Generador de descripciones de producto

- Sirve para: mejorar descripcion del catalogo.
- Entrada: nombre, categoria, precio, talles, imagen/contexto.
- Salida: descripcion premium breve.
- Prompt interno: "Copywriter ETER. Estetica sobria, urbana, premium. No inventes historia tecnica si no esta confirmada."
- Pantalla: Admin productos.
- Login: admin/soporte.

## Respuestas para clientes

- Sirve para: preguntas de talle, entrega, pago, disponibilidad.
- Entrada: pregunta + contexto de producto.
- Salida: respuesta de 2-3 oraciones.
- Prompt interno: "Responder claro, pedir talle/zona si falta, recomendar confirmar stock."
- Pantalla: Chat Aura, detalle, mensajes.
- Login: opcional para publico, completo para revendedor.

## Respuestas para proveedores

- Sirve para: pedir lista, imagenes, talles, disponibilidad, precio.
- Entrada: proveedor, objetivo, productos.
- Salida: mensaje/email.
- Prompt interno: "Mensaje operativo, breve, sin revelar datos internos innecesarios."
- Pantalla: Herramientas IA.
- Login: admin/soporte.

## Buscador interno de informacion ETER

- Sirve para: consultar reglas, stock, procesos, docs.
- Entrada: pregunta.
- Salida: respuesta con fuente o PENDIENTE_DE_CONFIRMAR.
- Prompt interno: "Responder solo con contexto oficial adjunto o datos conectados."
- Pantalla: Herramientas IA y Panel.
- Login: si para datos privados.

## Resumen de documentos

- Sirve para: resumir Docs/PDF/Drive.
- Entrada: archivo o texto.
- Salida: resumen, acciones y pendientes.
- Prompt interno: "Resumir para operacion ETER, separar hechos de pendientes."
- Pantalla: Drive/Archivos.
- Login: si.

## Herramientas de catalogo

- Sirve para: detectar productos sin imagen, stock bajo, duplicados, campos incompletos.
- Entrada: lista de productos.
- Salida: alertas y acciones sugeridas.
- Prompt interno: "Auditar catalogo ETER. No modificar datos sin confirmacion."
- Pantalla: Admin catalogo.
- Login: admin/soporte.
